what it does
[+]
Self-healing SQL
When a query fails or returns nothing, the agent diagnoses the error — wrong column, too-strict filter, dialect mismatch — and retries autonomously up to 3 times before surfacing the issue.
[+]
Multi-database, zero config
Paste any MySQL, PostgreSQL, or SQLite connection string. QueryMind reads the live schema on first connect and routes SQL to the right dialect automatically.
[+]
Bring your own key
Use your own Groq or Gemini API key (both free tier) for unlimited queries. No key? Use the hosted fallback within the daily budget — displayed live so you always know where you stand.
[+]
Conversation memory
Ask follow-up questions naturally. "Now filter that by region" works because the last 5 turns stay in context for every query in a session.
[x]
Read-only, always
Every query is validated before execution. INSERT, UPDATE, DELETE, DROP — none of them will ever run. Your data is safe by design, not policy.
